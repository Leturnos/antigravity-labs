# Antigravity Labs

Repositório de projetos pessoais voltados para aprendizado, experimentação e desenvolvimento de software utilizando o antigravity.

Aqui ficam jogos, ferramentas, APIs, automações e qualquer outra ideia que pareça interessante de construir.

## Objetivos

- Aprender novas tecnologias na prática
- Desenvolver projetos completos do início ao fim
- Explorar diferentes áreas da programação
- Construir um portfólio público
- Registrar minha evolução como desenvolvedor

## Estrutura

```text
games/      - Jogos e experiências interativas
backend/    - APIs, serviços e aplicações de servidor
ai/         - Projetos envolvendo inteligência artificial
tools/      - Ferramentas e utilitários
````
